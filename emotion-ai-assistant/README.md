# Emotion-Aware AI Assistant

This is my final year engineering project. It is a Streamlit-based web application that detects user emotions using webcam input and provides emotionally aware AI responses using Google's Gemini model.

## Features

- User login and signup
- Text-based AI chat
- Voice input using speech recognition
- Text-to-speech response
- Real-time facial emotion detection using webcam
- Multimodal interaction using voice and emotion
- SQLite database for user management
- Admin panel for managing users

## Technologies Used

- Python
- Streamlit
- OpenCV
- DeepFace
- Google Gemini API
- SQLite
- SpeechRecognition
- pyttsx3
